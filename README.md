1. 左侧栏「最近更新」列表（取最新6篇）
2. 真搜索（标题/摘要实时过滤，命中词高亮）
3. 按 category 分区显示（技术分享/心情随笔/思维日记 固定排在前面，其他分类自动排后面）

## 新增：开场仪式感动画（滚动展开相框）
- 进页面先看到一个带透视倾斜的深色平板相框叠在页面上方（纯装饰浮层，不裁切、不包裹真实内容）
- 往下滚动大约 0.9 屏的距离，相框逐渐摆正、放大、边框和圆角归零、周围暗角消退，最终完全消失
- 消失后即彻底移出渲染（`display:none`），不会再以任何形式影响页面操作
- 按你的要求：**没有做次数限制**，每次刷新/进入页面都会重新播放一次
- ⚠️ 已知的边界情况：如果博客文章数量很少、页面内容总高度不够一屏的 0.9 倍，可能来不及滚动完就到底部了，相框会停在半展开状态。等文章数量上来之后这个问题自然消失；如果现在就想避免，可以告诉我，加一段"内容不够高时自动跳过动画"的判断逻辑

## 再次更新：加入打字机标题 + 可见相框边框
- 相框改成参考图那种粗边框可见样式（不再是纯暗角遮罩），左上角加了 "// jerry.dev" 小水印，视觉上更接近你给的参考截图
- 屏幕上方新增打字机标题：小字"欢迎来到" + 大字逐字打出 "JERRY.DEV"，只播一次，不循环不删除
- 打字内容和相框会一直保留展示，直到滚动进度到 85% 才开始淡出（交叉淡出效果），滚动到 100% 时完全消失露出真实博客
- 滚动过程中相框本身也会从倾斜变平、从较小尺寸放大到接近整屏

## 第三次更新：底部漏半边 + 静态大字压叠 + 两阶段动画
- 背景改成纯黑
- 平板初始状态压在大字下方、底部裁在视口外（只露出上半部分），滚动后才整个升起来
- 外层大字改成静态展示（不再打字机），内容是"Jerry说：原神牛逼"，平板顶部边缘会压住文字底部一点点，做出图层叠压感
- 打字机效果挪进了平板"屏幕"内部，先打"欢迎来到jerry博客站"，停顿后打"我在打原神，有事请烧纸联系"，两行都只播一次不循环
- 动画拆成两个阶段：前65%滚动距离用来完成"平板升起+放大铺满全屏"，必须这个动作完全走完，后35%滚动距离才开始让整体淡出、露出真实博客——不再是一边放大一边淡出
