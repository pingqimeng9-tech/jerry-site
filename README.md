## 素材清单 + 摆放位置 + 尺寸建议

### 视频（4个，放进 `videos/`）

| 文件名 | 用在哪 | 建议规格 |
|---|---|---|
| `rumi-hero.mp4` | 首页开场大视频，铺满整个 hero 区背景 | 横版 16:9，建议 1920×1080，**必须以黑屏结尾**（你已经做好了）；文件别太大，建议控制在 10MB 内，不然移动端加载慢 |
| `reel-1.mp4` | "家长说/抖音精选"板块，第1个竖版视频卡片 | 竖版 9:16（抖音原生比例直接能用），最大显示高度 340px，会自动裁切填满 |
| `reel-2.mp4` | 同上，第2个 | 竖版 9:16 |
| `reel-3.mp4` | 同上，第3个 | 竖版 9:16 |

这3个 reel 视频是**静音循环播放**的（`muted loop`），点击卡片会带用户去你抖音主页看完整版，所以这里放**没有声音也能看懂的片段**（比如动画角色的可爱画面、孩子开口说英语的瞬间）效果最好。

### 图片（9张，放进 `images/rumi/`）

| 文件名 | 用在哪 | 建议尺寸 |
|---|---|---|
| `hero-poster.jpg` | 视频加载出来之前的占位图（首帧预览） | 1920×1080，选视频里一帧好看的画面截图即可 |
| `char-rumi.png` | 角色卡片1（主角 Rumi） | 正方形 1:1，建议至少 600×600，背景透明或纯色都行，会被裁成圆角方形 |
| `char-2.png` | 角色卡片2 | 正方形 1:1，同上 |
| `char-3.png` | 角色卡片3 | 正方形 1:1，同上 |
| `char-4.png` | 角色卡片4 | 正方形 1:1，同上 |
| `merch-plush.jpg` | 周边卡片：玩偶 | 正方形 1:1，建议至少 800×800 |
| `merch-stationery.jpg` | 周边卡片：文具 | 正方形 1:1 |
| `merch-cards.jpg` | 周边卡片：卡片/贴纸类 | 正方形 1:1 |
| `merch-sticker.jpg` | 周边卡片：贴纸 | 正方形 1:1 |

角色和周边这几张图**统一用正方形**，是因为对应的 CSS 卡片是 `aspect-ratio:1/1` 固定比例，长方形图片塞进去会被自动裁切（`object-fit:cover`），重要内容别贴太靠边缘，容易被裁掉。

### 教材板块——不需要图片
"同步教材"那一块（三本书堆叠的效果）是纯 CSS 画出来的卡片效果，不需要你传图，如果之后想换成真实书籍封面照片，跟我说一声，我把那部分代码改成读图片。

## 上传方式（GitHub 网页操作）

1. 进你的仓库，点 **Add file → Upload files**
2. 把 `index.html`、`rumi.html` 拖进去（会自动覆盖同名旧文件）
3. 再点一次 **Add file → Upload files**，这次把整个 `images/rumi` 文件夹拖进去
4. 再上传一次，把整个 `videos` 文件夹拖进去
5. 等 Vercel 自动重新部署（1-2分钟）
