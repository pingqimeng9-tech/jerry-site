# 博客更新包

## 这次改了什么
1. 侧边栏：头像下面新增「文章数 + 总浏览量」统计块 + 一排社交账号图标按钮
2. 背景换成图片壁纸（暗色蒙层+悬浮微光粒子），移除了之前的Simplex Noise流体画布
3. 文章详情页：用Notion文章封面图当页面背景；新增浏览量显示，每次打开+1
4. 评论功能：本轮不做

## ⚠️ 部署前必须做的事

### 1. Notion数据库加一个新属性
新建一个**数字（Number）**类型的属性，名字必须精确叫：`浏览量`
（不加这个属性，浏览量功能会静默失败，不影响文章正常显示，只是数字一直是0）

### 2. 上传壁纸图片
把你想要的壁纸图片传到：`images/blog-bg/wallpaper.jpg`
文件名必须完全一致（包括后缀.jpg）。传之前会显示纯色兜底背景，不会报错。

### 3. 改真实社交账号链接
`blog.html`里的 `.social-row` 那几个链接（GitHub/抖音/邮箱/RSS）目前是占位地址，
搜索 `★ 素材位` 能快速定位到，换成你真实的账号地址。

### 4. 文件覆盖
- `blog.html`、`post.html` → 覆盖仓库根目录同名文件
- `api/posts.js`、`api/post.js` → 覆盖 `api/` 目录同名文件
- `api/view.js` → 新增到 `api/` 目录
- `images/blog-bg/` → 新建这个文件夹，放你的壁纸图
